# Image Organization

This folder contains all images for the portfolio website, organized by purpose:

## Structure

```
/images
├── /home                    - Background images for the landing page
├── /projects               - Project-specific image folders
│   ├── /one-of-a-kind     - One of a Kind project images
│   ├── /women             - Women project images
│   ├── /twisters          - Twisters project images
│   ├── /frequencies       - Frequencies project images
│   └── /an-atomy          - An Atomy project images
└── /exhibitions           - Exhibition-specific image folders
```

## Image Guidelines

- All images should be monochrome (black and white)
- Use high-resolution images suitable for web display
- Name files descriptively
- Supported formats: PNG, JPG, WebP

## Usage

- **Home backgrounds**: Place 5+ images in `/images/home/` for random rotation
- **Project galleries**: Place project artwork in respective project folders
- **Exhibitions**: Create subfolders for each exhibition with gallery images
